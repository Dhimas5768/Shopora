Shopora - Premium E-Commerce HTML Template
Overview
Shopora is a modern, responsive, one-page e-commerce HTML template built with Tailwind CSS, Lucide Icons, and vanilla JavaScript. Perfect for fashion stores, shoe stores, electronic stores, cosmetic stores, gadget stores, and small marketplaces.

Features
✅ Fully Responsive Design
✅ Dark Mode (with localStorage persistence)
✅ Product Hover Animations
✅ Add to Cart Animation (flying to cart icon)
✅ Loading Screen
✅ Sticky Navbar with blur effect
✅ Smooth Scroll
✅ Product Badges (Sale, New, Hot)
✅ Wishlist Toggle (UI)
✅ Search Bar Overlay (UI)
✅ Category Filter with animation
✅ Product Quick View Modal
✅ Scroll Reveal Animations
✅ Flash Sale Countdown Timer
✅ Toast Notifications
✅ Back to Top Button
✅ Mobile Slide-in Menu
✅ Keyboard Support (ESC to close modals)

## Usage
1. Download/clone the template
2. Open `index.html` in your browser
3. Customize colors, products, and content as needed

## Note
This is a **frontend-only** template. All buttons (Add to Cart, Buy Now, Checkout, Sign In) are UI-only with no backend processing.
